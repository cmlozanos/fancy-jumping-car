# Assets

Carpeta de recursos visuales (sprites, fondos, tiles y UI). Todo en SVG para iterar rapido.

Estructura:
- sprites/: personajes, objetos y UI interactiva.
- backgrounds/: fondos de mundos/temas.
- tiles/: piezas de escenario reutilizables.

Nota: Los SVG usan estilo pixel-art con bloques rectangulares. Luego se pueden exportar a PNG si hace falta.
